# Stack

- React Native (apps/mobile/) (Depreciado não atualize mais este projeto)
- Next.js (apps/web)
- NodeJS com NestJS (apps/api)

# Instruções

- Sempre utilize o modo de planejamento, por mais simples que a tarefa seja
- Siga boas práticas de código e arquitetura, como código limpo
- Siga a documentação mais recente utilizando o MCP do context7
- Não utilize comentários a menos que seja necessário
- Quebre o código em arquivos menores sempre que possível, para evitar arquivos gigantes
- Utilize padrões já utilizados pelo mercado e boas implementações
- Não utilize comentários a menos que esteja descrevendo um código muito complexo
- Mantenha o padrão do Keep It Simple, ou seja, não faça complexidades desncessárias
- Utilize sempre versões atualizadas de bibliotecas ou frameworks

# Atualizações

- Atualize apenas o projeto web e a API, o projeto mobile está depreciado e será descartado
